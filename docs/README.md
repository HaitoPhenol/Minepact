# Minepact

A tiny AI-driven text world prototype.
A pact-bound spirit lives in your terminal, perceives a world, and talks.

**Status**: Early prototype. Command-line loop only.

## Setup

1. Please run `pip install openai` at first.
2. Copy `config_example.py` as a new file named `config.py` and fill in your API key.
3. Run `python main.py`.
